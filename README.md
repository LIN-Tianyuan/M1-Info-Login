# Login Page
## Install
This project uses python and flask framework.
```
pip install flask
pip install pymysql
```

## Database information
host = "localhost"

user = "root"

password = ""

database = "documents"

Database table name: user
Table Fields: 
 - id(int11) AUTO_INCREMENT
 - username(varchar20) 
 - password(varchar50)
## Usage

Run the login.py file

Then go to this website http://127.0.0.1:8080 to log in !
